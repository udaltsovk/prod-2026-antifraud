## Описание

Задание олимпиады по backend-разработке. Необходимо реализовать REST API сервис для обнаружения мошеннических транзакций. Сервис должен поддерживать JWT-аутентификацию, управление пользователями, обработку транзакций и настраиваемые правила антифрода на основе DSL (domain-specific language). Репозиторий содержит только спецификацию и документацию — код реализации нужно написать самостоятельно на любом языке программирования.

## Структура репозитория

- `en/` — документация и OpenAPI спецификация на английском языке
- `ru/` — документация и OpenAPI спецификация на русском языке
- `en/openapi.yml`, `ru/openapi.yml` — полная спецификация API
- `en/docs/`, `ru/docs/` — требования, DSL, FAQ, инструкции по запуску

## Description

Backend development olympiad assignment. The task is to implement a REST API service for detecting fraudulent transactions. The service must support JWT authentication, user management, transaction processing, and configurable anti-fraud rules based on a DSL (domain-specific language). This repository contains only the specification and documentation — the implementation code must be written independently in any programming language.

## Repository Structure

- `en/` — documentation and OpenAPI specification in English
- `ru/` — documentation and OpenAPI specification in Russian
- `en/openapi.yml`, `ru/openapi.yml` — complete API specification
- `en/docs/`, `ru/docs/` — requirements, DSL, FAQ, run instructions
